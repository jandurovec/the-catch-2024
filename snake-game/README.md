# Snake game (4 points)

Hi, TCC-CSIRT analyst,

snakes are wonderful creatures, and everyone loves them. Many people enjoy
playing with snakes and find them fascinating companions. One on them lives on
`snakegame.leisure.tcc` on port `23001/TCP`.

See you in the next incident!

## Hints

* The snake is actually a python.

## Solution

We can use `nc` to talk to the service on a provided port. Once the connection
is established, the server asks for code in Python, interprets it, prints a
result and closes the connection.

```console
$ nc snakegame.leisure.tcc 23001
Hello, I can only speak Python, show me your code.
Enter your code :
```

```python
os.environ
```
```text
This is not allowed: os
```
